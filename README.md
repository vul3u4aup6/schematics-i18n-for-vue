# Repository Description

This vue-repo can quick modify i18n at terminal.

### Installation

```bash
npm i i18n-lb-vue
npm install @phenomnomnominal/tsquery --save-dev
```

### How to use

```bash
schematics i18n-lb-vue:i18n
```

### Folder Structure

```bash
|-src
  |_assets
    |_lang
      |_tw.js
      |_zh.js
      |_en.js
      |_vi.js
```

That's it!

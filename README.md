# Repository Description

This vue-repo can quick modify i18n at terminal.

### Installation

```bash
npm i i18n-vue
npm i -g @angular-devkit/schematics-cli@0.13.4
npm install @phenomnomnominal/tsquery --save-dev
```

### How to use

```bash
schematics i18n-vue:i18n
```

### Folder Structure
src > lang > i18n > tw.js

That's it!
